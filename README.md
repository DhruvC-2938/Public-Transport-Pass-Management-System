# Public Transport Pass Management System (C++)

A **menu-driven C++ application** that simulates a **Public Transport Pass Management System**.  
The system allows users to create transport passes, check pass validity, and renew passes while automatically handling student discounts and pricing updates.

---

## Features

- Apply for a **new transport pass**
- Check **pass validity using Pass ID**
- Renew existing passes
- **Automatic student discount calculation**
- **Price updates during pass renewal**
- Menu-driven console interface
- Supports **multiple passes using arrays**

---

## Technologies Used

- **Programming Language:** C++
- **Concepts Used:**
  - Classes and Objects
  - Arrays
  - Loops
  - Conditional Statements
  - Switch Case
  - Menu-Driven Programming

---

## Stored Data Fields

Each transport pass stores the following information:

- Pass ID
- Holder Name
- Age
- Pass Type (Student / Regular)
- Pass Plan (Monthly / Quarterly / Half-Yearly / Yearly)
- Validity Days
- Price

---

## Program Workflow

1. Start the program  
2. Display the main menu  
3. User selects an option  
4. Apply Pass → Enter passenger details  
5. Check Validity → Search pass using Pass ID  
6. Renew Pass → Extend validity and update price  
7. Repeat the process until user exits
